# Visualización de un árbol rojinegro

Este proyecto requiere el uso de `make` para compilar, limpiar los archivos de compilación y ejecutar el programa.

## Compilar

Para compilar el proyecto, simplemente ejecuta el siguiente comando:

```bash
make
```

Este comando compilará el código fuente y generará los archivos ejecutables necesarios.

## Limpiar archivos de compilación

Para borrar todos los archivos generados durante la compilación, utiliza el siguiente comando:

```bash
make clean
```

Este comando eliminará los archivos objeto, ejecutables y cualquier otro archivo temporal generado durante la compilación.

## Ejecutar el programa

Para ejecutar el programa, usa el siguiente comando:

```bash
./main < datafile
```

Donde `datafile` es un archivo de texto que contiene una clave en cada línea. El programa leerá estas claves y las procesará según la lógica implementada.
